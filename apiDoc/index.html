1. 认证相关API

POST /api/auth/login

用户登录

用户登录接口，返回token和用户信息

请求参数

参数名 类型 是否必填 描述
username string 是 用户名
password string 是 密码

响应示例

{ "code": 200, "message": "登录成功", "data": { "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "user": { "id": "1", "username": "test", "nickname": "测试用户", "email": "test@example.com", "avatar": null, "role": "user", "status": "active" } } }
POST /api/auth/register

用户注册

用户注册接口，创建新用户

请求参数

参数名 类型 是否必填 描述
username string 是 用户名
password string 是 密码
nickname string 否 昵称
email string 否 邮箱

响应示例

{ "code": 200, "message": "注册成功", "data": { "id": "2", "username": "newuser", "nickname": "新用户", "email": "newuser@example.com" } }
POST /api/auth/logout

用户登出

用户登出接口，清除token
认证要求: 需要在请求头中添加 Authorization: Bearer {token}

响应示例

{ "code": 200, "message": "登出成功" }

7. AI解析相关API

POST /api/ai/analyze

生成AI解析

save boolean 否 是否自动保存解析结果

响应示例
